# SocialMedia 

Socialmedia Webpage made with:
* PHP,
* HTML,
* CSS,
* Bootstrap,
* jQuery.

## Demo

Demo avialable at Demo[http://socialmedia-mt.cba.pl/]

## Features

* Register / login,
* Posts on own / other users table,
* Comment posts,
* Messages (i.e. chat),
* Likes, 
* Notificaitons (other users activities),
* Account settings (change profile details / picture),
* Add / remove friends,
* Search for users,
* Fully responsive
